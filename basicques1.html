What is internet ?

The internet is a global system of interconnected computer networks that use the internet protocol to communicate between networks and devices

Example:sending mails,social media,etc...

What is browser ?

A browser also known as a web browser,is a software application that allows users to access and view content on the World Wide Web

Example:Google Chrome, Mozilla Firefox, Safari, and Microsoft Edge

What is URL ?

A URL (Uniform Resource Locator, also called a web address) is a unique identifier used to locate a resource on the internet

Example: https://www.google.com

What is http ?

HTTP(Hypertext Transfer Protocol) is the foundational protocol that enables communication between web clients and servers

What is https ?

HTTPS stands for Hypertext Transfer Protocol Secure. It's the secure version of the standard HTTP protocol used on the internet
